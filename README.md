# ETF多因子MACD评分模型

全市场ETF扫描，6因子打分，评分≥4.0买入。

## 策略概要

| 规则 | 参数 |
|---|---|
| 入场 | MACD金叉 + 6因子评分≥4.0 |
| 买入 | T+1开盘价（跳空>2%不买） |
| 止损 | -8% |
| 持有 | 30天到期 |
| 冷却 | 15天 |
| 仓位 | 单票≤30% |

### 6因子

| 因子 | 满分条件 |
|---|---|
| F1 MACD金叉 | 必要条件 |
| F2 趋势 | 价格>MA200 (+0.5), MA50>MA200 (+1.0) |
| F3 量能 | 成交量>1倍均量(+0.5), >1.5倍(+1.0) |
| F4 非V反 | 不在V型反转中(+1.0) |
| F5 柱强度 | MACD柱/价格>1%(+0.5), >2%(+1.0) |
| F6 低波动 | ATR%<3%(+0.5), <2%(+1.0) |

满分6.0，≥4.0触发买入。

### 回测表现（全市场173只ETF，2016-2026）

|---|---|
| 总交易 | 1,648笔 |
| 胜率 | 52.7% |
| 均笔收益 | +2.99% |
| 止损率 | 25.4% |
| 总收益 | +4,927% |

## 环境要求

- Python 3.11+
- MySQL 8.0（存放ETF历史数据）
- 网络（拉取实时行情）

## 快速开始

### 1. 克隆

```bash
git clone git@github.com:tongtongtong2/-.git
cd -/etf_trader_v2
```

### 2. 安装依赖

```bash
pip install pymysql
```

### 3. 配置MySQL

编辑 `etf_trader_v2/config.py`，修改数据库连接：

```python
MYSQL_HOST = "127.0.0.1"
MYSQL_PORT = 3306
MYSQL_USER = "root"
MYSQL_PASSWORD = "your_password"
MYSQL_DB = "etf_trader"
```

如果办公本没有MySQL，需要先导入历史数据。数据文件较大，单独提供。

### 4. 运行

**收盘后扫描（确认信号，用实际收盘价）：**
```bash
cd etf_trader_v2
python daily_score_scan.py
```

**2:50实时扫描（用当前价估算，提前10分钟）：**
```bash
python daily_score_scan.py --realtime
```

## 每日操作流程

1. **2:50** — 运行 `python daily_score_scan.py --realtime`
2. 看输出：有评分≥4.0的金叉信号 → 准备下单
3. 收盘后 — 运行 `python daily_score_scan.py` 确认
4. 第二天开盘 — 检查是否跳空>2%，不超就买入
