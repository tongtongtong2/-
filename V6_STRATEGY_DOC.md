- 熊市：收盘价 < MA60 → 持仓2只

## 三、选股评分函数

### 硬过滤（不通过直接排除）：
1. 股价 < 3元 → 排除
2. 今日涨停（涨幅>9.5%）→ 排除（不追涨停）
3. 5日涨幅 > 12% → 排除（过热）
4. 5日涨幅 < -3% → 排除（弱势）
5. 20日涨幅 > 30% → 排除（过热）
6. 20日均量 < 3000手 → 排除（流动性差）
7. 5日/20日量比 < 0.8 → 排除（缩量）
8. 收盘价 < MA5 → 排除
9. 收盘价 < MA20 → 排除
10. MA5 < MA20 → 排除（短期趋势弱）
11. 20日收益率标准差 > 4% → 排除（波动太大）
12. 上影线 > 1.5倍实体 → 排除（出货形态）

### 评分公式：
momentum = (ret5×2 + ret10 + ret20×0.5) / 3.5
score = momentum×0.45 + min(vol_ratio, 2.5)/2.5×0.20 + trend×0.35

其中：
- ret5/ret10/ret20 = 5/10/20日收益率
- vol_ratio = 5日均量/20日均量
- trend = (0.5 if MA20>MA60 else 0) + (0.5 if MA5>MA20 else 0)

### 出货信号检测（通过筛选后额外标记）：
- 长上影线：上影 > 2倍实体
- 高开低走：阴线且实体占振幅>60%
- 放量滞涨：今日量>前4日均量×1.5 且涨幅<1%

## 四、仓位管理
- 每只仓位 = 总资产 / max_positions（牛市5/中性3/熊市2）
- 每天买1只，N天建满仓
- 买入价：开盘价附近

## 五、止盈止损
- 固定止损：买入价 × 0.93（亏7%走人）
- 移动止盈：涨8%激活，之后最高价×0.97跟踪（回撤3%卖）
- 超时卖出：持仓满20个交易日强制卖出

## 六、回测引擎逻辑（伪代码）
```
for each trading_day:
    # 1. 检查持仓止损/止盈/超时
    for pos in positions:
        if today_low <= pos.stop_loss: sell(pos, stop_loss_price)
        if pos.trail_active and today_low <= pos.trail_sell: sell(pos, trail_sell)
        if not pos.trail_active and today_high >= pos.trail_activate:
            pos.trail_active = True
            pos.highest = today_high
        if pos.trail_active and today_high > pos.highest:
            pos.highest = today_high
            pos.trail_sell = pos.highest * 0.97
        if holding_days >= 20: sell(pos, today_close)
    
    # 2. 判断大盘状态，确定max_positions
    state = get_market_state(index_data)
    
    # 3. 如果有空仓位，选股买入
    if len(positions) < max_positions:
        candidates = score_all_stocks(today)
        buy top1 candidate at today's close price
```
